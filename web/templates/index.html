How it works

  1. Fill in Profile. Add company roles under Work & Internships, or student projects under Projects (section 4 shows up to 3 entries; work/internships first).
  2. Add Abilities and Software.
  3. Open Figma Layers and name text layers in your template.
  4. Generate Resume: paste a job description → get resume.json + Figma JSON.
  5. Copy the Figma JSON into the plugin (or run generate.py in the terminal).
