# cvBuilder 数据格式准则（固定信息 & 日期）

全系统统一遵守。资料库录入（`onboard.py`）、项目录入（P1）、简历生成（`generate.py`）均引用本文件。

**原则**：格式不对 → 记入警告/报告，**不中断流程**；能写入的照常写入，缺失项最后提醒用户补全。

---

## 1. 语言无关字段（只存一份）

| 字段 | 格式 | 示例 |
|------|------|------|
| `name` | 官方拼写，拉丁字母 | `Yichen Wang` |
| `contact.email` | 小写邮箱 | `name@example.com` |
| `contact.address` | 街道, PLZ 城市 | `Kantstrasse 9A, 97074 Würzburg` |
| `school` | 学校官方名 | `Universität Würzburg` |

### 1.1 电话号码 — DIN 5008（E.123 国际格式）

`contact.phone` 遵循 [DIN 5008](https://de.wikipedia.org/wiki/DIN_5008) 电话书写规范（ITU E.123 国际表示法）。

| 规则 | 说明 |
|------|------|
| 国际冠字 | 用 `+` + 国家码，**不用** `0049` |
| 区号/前缀 | 国家码后**不加**国内长途 `0`（`+49 931` 而非 `+49 0931`） |
| 分隔 | 各号段之间用**空格**，不用 `/`、`.` 作主分隔 |
| 分机 | 主号与分机用 ` - ` 或 ` Durchwahl `，如 `+49 931 1234567 - 42` |
| 禁止 | `(0)` 括号、无空格连写 `+4915206820364`、纯国内 `0152...` 存入 |

**示例（德国）**

```
+49 132 06345678          # 手机
+49 931 1234567           # 固话 维尔茨堡
+49 30 12345678            # 固话 柏林
+49 911 1234567            # 固话 纽伦堡
```

**录入转换**：`0152 06345678` / `0049 (0)152/06345678` → 规范为 `+49 152 06820364`。无法识别国家码时保留原值并警告。

## 2. 双语字段（必须 de + en）

`degree`、`languages[].name/level`、`summaries[].text`、`application_context.note`、项目 `title/role/bullets` 等。

用户用任意语言输入 → 系统整理为 `{ de: "...", en: "..." }`。

---

## 3. 日期格式（德式 · 日月年）

**所有 `period` 字段**（教育、项目、实习）统一用德式 **Tag.Monat.Jahr**，不使用 ISO `YYYY-MM-DD` 或 `YYYY-MM` 作为最终存储格式。

### 3.1 分隔符

- 起止范围：`开始 - 结束`（空格 + 短横线 + 空格）
- 进行中结束写：`heute`（小写）

### 3.2 精度层级（按信息量选用）

| 精度 | 格式 | 示例 |
|------|------|------|
| 精确到日 | `TT.MM.JJJJ` | `01.04.2025` |
| 仅到月 | `MM.JJJJ` | `04.2025` |
| 德语月份名 | `Monat JJJJ` | `April 2025` |

### 3.3 范围示例

```
01.04.2025 - heute
04.2025 - 07.2025
Oktober 2023 - Oktober 2024
11.2025 - 03.2026
```

### 3.4 转换规则（录入时）

用户输入 `2025-04-01`、`2025/4/1`、`April 1 2025` 等 → 规范为 `01.04.2025` 或 `April 2025`（有日则 TT.MM.JJJJ，无日则 MM.JJJJ 或 Monat JJJJ）。

**不完整 period**（空字符串、只有开始无结束且无 `heute`）：允许存入，但在报告中标 TODO；生成简历时**跳过该条**，不报错。

### 3.5 `application_context.earliest_start`

实习可开始时间：`MM.JJJJ`，如 `08.2026`。

## 4. 缺失字段策略

| 情况 | 行为 |
|------|------|
| 缺少邮箱/电话等 | 写入空字符串 `""`，末尾警告提醒补全 |
| 缺少教育/项目 period | 保留条目，period 为 `""`，报告标 TODO |
| 缺少 de/en 一侧翻译 | 另一侧照常；报告标 TODO，不现场编造 |
| LLM 响应缺 `cover_letter` | 写入占位说明，警告提醒 |
| `output_language` 无效 | 默认 `en`，警告提醒 |

**只有**完全无法解析（无 `profile` / 无 `resume`）时才终止。
